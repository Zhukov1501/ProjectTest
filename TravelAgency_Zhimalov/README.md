# Турагентство - Система управления

## Описание проекта
WPF приложение для турагентства с интеграцией данных парка развлечений.

## Структура проекта
```
TravelAgencyApp/
├── Models/           - Модели данных
├── Views/            - Окна WPF
├── ViewModels/       - ViewModels (MVVM)
├── Services/         - Бизнес-логика
├── Data/             - Работа с базой данных
├── Converters/       - Конвертеры данных
└── TravelAgency.db   - SQLite база данных
```

## Задания

### Задание 6: Авторизация
- Форма входа с логином и паролем
- Три роли пользователей: Admin, Manager, User
- Показ/скрытие пароля
- Защита паролей с BCrypt

### Задание 8: Услуги парка развлечений
- CRUD операции для услуг
- Категории: Развлечения, Экстрим, Семейные, Детские, Технологии, Водные, Кино, Спорт, Комплекс
- Цена, длительность, вместимость, локация

### Задание 11: Поиск, сортировка и фильтрация
- Поиск по названию, описанию, локации
- Фильтрация по категории
- Фильтрация по диапазону цен
- Сортировка по всем полям (ASC/DESC)

### Задание 18: Клиенты турагентства
- CRUD операции для клиентов
- Паспортные данные, контакты
- История регистрации

## Запуск
```bash
cd TravelAgencyApp
dotnet run
```
